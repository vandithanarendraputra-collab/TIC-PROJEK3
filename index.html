Kebijakan Cookie HEALERGY

Apa itu Cookie?

Cookie adalah file teks kecil yang disimpan di perangkat Anda untuk meningkatkan pengalaman browsing Anda.

Bagaimana Kami Menggunakan Cookie?

Kami menggunakan cookie untuk mengingat preferensi Anda, seperti pengaturan bahasa dan mode tampilan (Dark/Light Mode), serta untuk memudahkan proses login simulasi.

Pihak Ketiga

Beberapa layanan simulasi pihak ketiga (seperti simulasi Login Google/Apple) mungkin menggunakan cookie mereka sendiri untuk fungsionalitas teknis.

Mengelola Cookie

Anda dapat mengatur atau menghapus cookie melalui pengaturan browser Anda kapan saja. Namun, beberapa fitur website mungkin tidak berfungsi maksimal tanpa cookie.
